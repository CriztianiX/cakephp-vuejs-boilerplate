This bolierplate contains
 * PHP8
 * CakePHP 4.x
 * Vuejs 3
 * cakephp/twig-view

Running the application
```bash
docker-compose up
```

Install npm libraries
```bash
$ cd app/
$ npm install
```

Compile Vue application
```
$ cd app/
$ npm run development
```

Test your environment
```
Open your browser, head to http://127.0.0.1:9090/home/index
```
